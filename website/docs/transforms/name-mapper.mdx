Create a name by mapper.

:::note
Using this transform available only with another name-transform, example with **name/cti/kebab**.
:::


## Usage

Configure tool:

```json title="themekit.config.json" {7}
{
  "entry": {
    "default": "./themes/default.theme.json"
  },
  "output": {
    "css": {
      "transforms": ["name/cti/kebab", "name/mapper"],
      "buildPath": "./themes",
      "files": [
        {
          "destination": "[entry]/root.css",
          "format": "css/variables"
        }
      ]
    }
  }
}
```

Configure theme with mappers:

```json title="themes/default.theme.json" {3}
{
  "sources": ["./themes/tokens/*.tokens.yml"],
  "mappers": ["./src/theme/legacy.mapper.json"]
}
```
